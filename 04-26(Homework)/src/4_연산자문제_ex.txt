[연산자 확인문제]
제한시간 - 20분

실행용 클래스 : kh.java.operator.TestMain.java
기능제공용 클래스 : kh.java.operator.TestOperator.java
각 문제별 메소드명은 각자 임의로 작성하십시오.

1. 사용자에게 세자리 이상의 숫자를 입력 받아 백의 자리 이하를 버리는 코드를 작성해 보십시오.
	출력 예1) 숫자 입력 : 337
		  결과 : 300

	출력 예2) 숫자 입력 : 1274
		  결과 : 1200

2. 사용자에게 두개의 숫자를 입력 받습니다
   (첫번째 숫자 : 총 과일의 갯수, 두번째 숫자 : 한개의 바구니에 들어가는 과일의 갯수)
   과일을 모두 담으려면 몇개의 바구니가 필요한지 확인하는 코드를 작성하시오.
	출력 예) 과일 갯수 : 27
		 바구니 과일 수 : 5
		 27개의 과일을 담으려면 6개가 필요합니다.
***확장판 : 마지막 바구니에 들어가는 과일의 수를 추가로 출력 하시오.


3. 영어 하나의 문자를 입력하여 대문자는 소문자로, 소문자는 대문자로 변경하는 코드를 작성하시오.
   (대문자와 소문자는 32만큼 차이납니다)
	출력 예)문자 입력 : A
	        A -> a
